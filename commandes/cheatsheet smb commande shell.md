
•**ls** : affiche la liste des fichiers et des répertoires dans le répertoire courant du partage.
• **cd**: modifier le répertoire courant du partage.
•**get** : télécharge un fichier depuis le partage vers votre système local.
•**put** : envoie un fichier depuis votre système local vers le partage.
•**mkdir** : crée un nouveau répertoire dans le répertoire courant du partage.
•**rmdir** : supprimé un répertoire vide dans le répertoire courant du partage.
•**rm** : supprimer un fichier ou un répertoire (et son contenu) dans le répertoire courant du partage.
